Did you want to use bytebin for your app?

Depending on what you're after, the pastes.dev 'official' service may be a good fit. Please have a read the API documentation and the Terms of Service, and proceed with that if it is!

Otherwise, please self-host your own instance of bytebin. The bytebin GitHub page has some basic instructions.

Please do not use this (https://bytebin.lucko.me) service.

Background

This service (https://bytebin.lucko.me) was originally created in 2018, and was used primarily for my own personal projects. Some others asked for permission to use it too, and I generally allowed that on a case-by-case basis. Many others have also used it without permission. Fast-forward to today, my personal projects are using different solutions, and this service is now defunct.

It only exists today for backwards compatibility with some of my older projects, and to allow users time to migrate to other solutions. It will be permanently turned off soon. In the meantime, if you come across any projects that are still using this service or referencing the https://bytebin.lucko.me URL, please consider letting the author know that they should migrate to a different solution.
